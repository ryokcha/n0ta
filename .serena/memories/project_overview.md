# n0ta Portfolio Project Overview

## Project Purpose
This is a portfolio website for Nakai Ryoka (中井涼日). The site showcases professional work and personal background.

## Project Type
Portfolio/Personal Website

## Tech Stack
- **Framework**: Next.js 16.0.1 with TypeScript
- **Styling**: Tailwind CSS 4
- **CSS Processor**: PostCSS
- **Runtime**: Node.js / React 19.2.0
- **Package Manager**: npm
- **Linting**: ESLint 9 with Next.js configuration

## Key Configuration
- **Language**: TypeScript (strict mode enabled)
- **Module Resolution**: Bundler
- **Import Alias**: `@/*` → root directory
- **CSS**: Tailwind CSS with PostCSS
- **Target**: ES2017
- **App Router**: Enabled (App directory structure)
